# Es. n.5 - Comunicazione TCP

Il progetto ha lo scopo di permettere la comunicazione tra server e client, tramite l'uso dei metodi Scrivi e Leggi

## Instalazione

## Scenari di utilizzo

#### 1) Avvio server e successivo avvio client:  
 Il client viene connesso al server senza errori e permette lo scambio di messaggi fino al messaggio di fine che dve essere inviato sia dal client che dal server

![Avv Client](img/AvvioClient1.png)

 In questo caso il client proverà a connettersi a un server inesistente, quindi il risultato sarà un messaggio di errore
